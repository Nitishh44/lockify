# 🔐 Lockify - Secure Password Manager

Lockify is a full-stack password manager built using **Node.js, Express, MongoDB, and EJS**.  
It allows users to securely store and manage their credentials with encryption and authentication.

---

## 🚀 Features

- 🔑 User Authentication (Signup/Login)
- 🔒 AES Encrypted Password Storage
- 👁️ Show / Hide Password Toggle
- 📋 Copy Password Functionality
- 🗑️ Delete Confirmation Modal
- ⏳ Auto Logout After Inactivity
- 🔍 Search Passwords
- 🎨 Modern Responsive UI
- 🟢 Status Indicators (Active/Inactive)
